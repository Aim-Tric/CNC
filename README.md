# 觅课网

基于Java Web技术，开发的一个课程开辟与线下面对面教学的网站。

## 一、项目目的

1. 为教练等有余力的人提供兼职平台
2. 为想学一项技能但无处可寻的人提供平台
3. 锻炼组员们在校期间的编程能力，开发项目能力，提升个人技能发展空间。

## 二、使用环境

代码托管：https://github.com/Aim-Tric/CoachAndCourse

运行服务器：Tomcat-9.0

通用IDE：ItelliJ Idea 

库类：jsp+(预定 SpringMVC)+mybatis+mysql
